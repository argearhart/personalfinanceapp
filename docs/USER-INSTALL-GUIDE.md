# Ledger & Balance: Install Guide (For End Users)

This guide is for people who just want to install and use the app.

There are two common ways this app is distributed. Your helper should tell you which one you have.

## Before You Start

### Option A: Web install (Chrome / Edge PWA)

- Works on **Windows, macOS, or Linux** (as long as your browser supports installable web apps).
- Use **Chrome** or **Microsoft Edge**.
- You need internet access to **install** and for **updates** (unless your helper hosts it on your local network).

### Option B: Offline Windows desktop installer (recommended for “no website” setups)

- Windows **64-bit** PC.
- No Chrome install required for daily use.
- Internet is **not** required after installation.
- Updates are **not automatic**: your helper will provide a new installer when needed.

## Option A: Install in 3 Steps (Chrome / Edge)

1. Open this link in Chrome or Edge:  
   `https://your-app-url-here`
2. In the browser address bar, click **Install** (or **Install app**).
3. Confirm install. The app will appear in your **Start Menu/Applications** and can be pinned to Desktop/Taskbar.

## Option B: Install in 3 Steps (Windows installer)

1. Double-click the installer your helper gave you (example name): `Ledger & Balance Setup x.y.z.exe`
2. Follow the prompts (one-click installer).
3. Launch **Ledger & Balance** from the Start Menu or Desktop shortcut.

## Open the App Later

### Option A (Chrome / Edge installed app)

- Windows: Start Menu -> search for **Checkbook** (PWA `short_name`) or **Ledger & Balance Checkbook** (PWA `name`)
- macOS: Applications -> your installed web app entry
- Linux: Applications menu -> your installed web app entry

### Option B (Windows desktop installer)

- Windows: Start Menu -> **Ledger & Balance**

## Updating the App

### Option A (hosted web app)

- Updates are automatic when you open the app while connected to the internet.
- If you do not see recent changes, close and reopen the app.

### Option B (offline desktop installer)

- Updates are **manual**.
- Your helper will provide a newer installer when it is time to update.

## Your Data and Backups

- Your data is stored **locally on your device**.
- Data does **not** automatically sync between computers.
- To move data to another computer:
  1. In the app, use **Export Database**.
  2. On the other computer, install the app and use **Import Backup**.

## Troubleshooting

### Option A (Chrome / Edge)

- **No Install button visible**
  - Use Chrome or Edge.
  - Refresh the page.
  - Make sure the URL starts with `https://`.
- **Install worked but app won't open**
  - Restart browser/computer and try again.
- **Data missing**
  - Confirm you are on the same computer/profile.
  - Re-import your backup JSON if needed.

### Option B (Windows installer)

- **Installer will not run**
  - Confirm you are on Windows 64-bit.
  - Try re-copying the installer from USB (file may be incomplete).
- **App opens but looks blank**
  - Fully quit the app and reopen.
  - If it persists, contact your helper (they may need to rebuild/reinstall).

## Need Help?

Share these details with support:

- Browser name/version (**Option A** only)
- Installer file name (**Option B** only), if applicable
